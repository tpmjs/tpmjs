# @tpmjs/tools-hardening-checklist-web

## 0.1.0

### Initial Release

- Initial implementation of Web Security Hardening Checklist tool
- Evaluates 22 critical security controls across 11 categories
- Calculates security score and assigns letter grade (A+ to F)
- Provides prioritized recommendations based on missing controls
- Categorizes items by priority (critical, high, medium, low)
- Includes impact descriptions for each security control
