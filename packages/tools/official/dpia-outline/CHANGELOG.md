# @tpmjs/tools-dpia-outline

## 0.1.0

### Initial Release

- Initial implementation of DPIA Outline tool
- Generates comprehensive DPIA document template following GDPR Article 35 requirements
- Identifies privacy risks based on data types and processing activities
- Provides risk severity and likelihood assessments
- Suggests detailed mitigation strategies with specific controls
- Detects high-risk processing requiring mandatory DPIA
- Includes all standard DPIA sections: legal basis, necessity, proportionality, data subject rights, etc.
